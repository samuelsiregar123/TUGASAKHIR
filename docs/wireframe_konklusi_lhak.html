Penjelasan untuk Anda (Samuel):
  • Satu menu, dua bagian. Di sidebar hanya ada satu menu "Konklusi & LHAK". Di dalamnya bagian atas untuk konklusi, bagian bawah untuk generate LHAK — tidak ada menu "Konklusi" terpisah.
  • Bagian 1 - Konklusi: 3 kartu (TK/MK/FK) dengan warna sesuai hasil — hijau Memadai, kuning Perlu Peningkatan, merah Tidak Memadai. Di bawahnya konklusi keseluruhan. Tombol "Hitung Ulang Konklusi" untuk menghitung dari nilai penilaian auditor.
  • Bagian 2 - LHAK: tombol "Generate LHAK" membuat PDF 7 halaman, lalu muncul preview dan tombol unduh.
  • Catatan FK: contoh ini menunjukkan FK "Tidak Memadai" dengan "EIK dilewati" — sesuai aturan jika EDK Tidak Memadai maka EIK tidak dinilai.
  • Beri tahu jika ingin tata letak, warna, atau label diubah.
